# Agent targets

ai-pkgs supports 47+ AI coding assistants. Each target maps to a project
skill directory and a global skill directory. `skills add` writes copies (or
symlinks with `--link`) into the chosen target's directory.

## Common targets

- `cursor`: project `.cursor/skills`, global `~/.cursor/skills`
- `claude-code`: project `.claude/skills`, global `~/.claude/skills`
- `codex`: project `.codex/skills`, global `~/.codex/skills`
- `universal`: project `.agents/skills`, global `${XDG_CONFIG_HOME:-~/.config}/agents/skills`

## Plugin-capable agents

Only three agents currently support plugins:

- `claude-code`: cache at `~/.claude/plugins/cache/`, settings at
  `~/.claude/settings.json` (global) or `.claude/settings.json` (project).
- `cursor`: shares the Claude plugin cache on macOS / Linux; uses
  `~/.cursor/extensions/` on Windows.
- `codex`: cache at `~/.codex/plugins/cache/`, registers each plugin in
  `~/.codex/config.toml` and `~/.agents/plugins/marketplace.json`.

See [plugins/add](./plugins/add.txt) for the target selector and
[plugins/targets](./plugins/targets.txt) for per-plugin target management.
